Statistiky

Obecně

Statistiky běhu jednotlivých jobů se nacházejí ve rozbalovacím menu Jobs. Po zmačknutí, menu se rozbalí a objeví se pod-menu Statistics:
Aktualně existuje 6 druhů statistik:
  • Actual - zobrazuje průběh jobů za poslední tyden
  • Full harvest - zobrazuje statistiky posledních jobů kompletního sklízení
  • Index all records - zobrazuje statistiky posledních jobů indexaci
  • Deduplicate records - zobrazuje statistiky posledních jobů deduplikaci
  • Download and import configurations - zobrazuje statistiky posledních jobů indexaci importovaní záznamů
  • Regenerate dedup keys - zobrazuje statistiky posledních jobů přegenerování klíčů
Statistiky se nacházejí ve vlastních složkách a pro zobrazení statistik konkretního jobu je potřeba zvolit jednu ze složek:
Vše statistiky obsahují následující informace:
  • ID - identifikátor vykonání jobu
  • Start - čas začátku jobu
  • End - čas konce jobu
  • Status - status jobu(COMPLETED, FAILED, STARTED)
  • Duration - doba běhu jobu kdy končíl
Okaždem jobu se dá získat podrobnější informace kliknutím šipky ve sloupce Details.
Také některé statistiky, kromě informací uvedených výš, uvádějí specifické informace relevantní pro konkretní job.
Statictiky o Full harvest jobu uvádějí:
  • Conf. ID - což je identifikátorem konfigurace která se používala pří sklízení
  • Lib. name - zkrátka názvu knihovný ze které se provádělo sklízení
  • URL - odkaz na WEB stránku knihovny
  • From/To - určují interval ve kterém by se měla provést aktualizace záznamu
  • Records - počet sklízených záznamů
Statistiky o jobu Index all records navic uvádějí adresu solr Solr URL.
U statistik o jobu Download import configurations je informace o formátu konfigurace.

Filtrování

Vše statistiky se dá filtrovat podle jejích statusů, statusy se zaškrkovají ve rozbalovacím menu:
Také lze provádět pokročilé vyhledávání prostřednictvím zadání parametrů Start, End, From a To:
Pro nastavení parametrů pro pokročilé vyhledávání je potřeba zmačknout symbol kalendáře, po zmačknutí se otevře okno s kalendářem kde lze nastavit odpovídající parametr:

V tomto okně lze provádět nastavení datu a času. Pro rychlejší nastavení času lze zmáčknout na hodnotu hodin(resp. minut) a otevře se panel s volbou hodin(resp. minut):

Pro mazání času lze použít křížek sleva od tlačítka kalendáře.
Pro provedení vyhledávání s pokročilým vyhledáváním je potřeba zmačknout tlačítko Search.
Pro zobrazení posledních 10 statistik je potřeba smazat hodnoty všech parametrů (Start, End, From, To) a zmáčknout Search.
